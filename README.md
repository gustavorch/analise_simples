# Análise Exploratória de Dados sobre Saneamento Básico no Brasil

## Descrição

Este projeto consiste em uma análise exploratória simples realizada a partir de um conjunto de dados contendo informações relacionadas ao saneamento básico nos estados do Brasil. O objetivo principal foi explorar os dados e identificar padrões ou relações entre algumas variáveis por meio de técnicas básicas de análise de dados.
